Getting Started with NGW

This guide explains how to set up your Angular project to begin using Ngw. It includes information on installing and importing Ngw to your Angular Application.

Install Ngw

Ngw Comes has a npm package so you can start installing Ngw by running following command :

npm i ngw-core --save

The above command installs Ngw and adds it has a dependency to your application.
