Ngw Typography

The text component is a simple component that can be used to style the text color of any element.

API
Directive Description Allowed Values
ngwTypo Defines style for text headline, title, subheading-2, subheading-1, body-2, body-1, body, caption, button, display-4, display-3, display-2, display-1
muted Defines if the text need to be muted true or false
